# ☁️ Microsoft Azure Labs | Hands-On Practice & Projects

## 📌 Overview
This folder documents my **hands-on Azure lab practice**, focused on building real-world cloud skills through guided and self-directed exercises.

The goal of this section is to demonstrate practical experience with:
- Azure core services
- Identity & Access Management (IAM)
- Cloud networking
- Security configurations
- Resource deployment & management
