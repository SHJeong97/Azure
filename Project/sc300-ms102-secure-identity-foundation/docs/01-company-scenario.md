# Company Scenario

## Business Overview
Northstar Health Advisors is a 75-user professional services company using Microsoft 365 for email, collaboration, file sharing, and identity.

## Departments
- Executive
- HR
- Finance
- Sales
- Operations
- IT

## Current Problems
- User onboarding is manual and inconsistent.
- Some administrators hold more privilege than they need.
- License assignment is done manually, causing waste and inconsistency.
- Remote users need stronger sign-in controls.
- Password reset issues generate unnecessary help desk work.
- Access removal during offboarding is too dependent on manual action.

## Business Requirements
- Standardize onboarding and access assignment.
- Reduce standing administrative privilege.
- Enforce stronger authentication for remote and cloud access.
- Improve license consistency with group-based assignment.
- Reduce password-reset support burden.
- Make validation and audit evidence easy to produce.

## Success Criteria
- New users can be assigned a standardized access pattern by department.
- Admin privileges are scoped and documented.
- Authentication controls are clearly defined and testable.
- License assignment is consistent and supportable.
- Validation evidence exists for every major control.
