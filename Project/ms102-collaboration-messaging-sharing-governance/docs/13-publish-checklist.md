# Publish Checklist

## Security Review
- No passwords stored in the repository
- No guest invite tokens or sensitive links exposed
- No browser session data exposed in screenshots
- No private external email information exposed beyond what is safe for lab use

## Content Review
- README section order is correct
- Evidence filenames referenced in the README exist
- Validation notes exist for each build phase
- Usernames, group names, mailbox names, Team names, and site names are consistent
- Project title is consistent across files

## Presentation Review
- Architecture diagram exists
- Business value file exists
- Interview talk track exists
- Evidence index exists
- Repo name is clean and professional
