# Exchange, Teams, and SharePoint Scope

## Exchange Online Scope
- shared mailboxes
- mailbox delegation
- distribution groups or mail-enabled collaboration paths
- basic mail flow governance

## Teams Scope
- department team creation
- membership validation
- collaboration structure validation

## SharePoint Scope
- department site ownership
- external sharing baseline
- site-level governance validation

## Admin Scope
- Microsoft 365 admin center
- Exchange admin center
- Teams admin center
- SharePoint admin center
