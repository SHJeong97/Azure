# Admin Role Model

## Planned Role Design

| Role | Purpose |
|---|---|
| Global Administrator | emergency and tenant-wide recovery only |
| Exchange Administrator | messaging and mailbox administration |
| SharePoint Administrator | SharePoint and OneDrive administration |
| Teams Administrator | Teams administration |
| User Administrator | user and license administration |
| Global Reader | read-only oversight |

## Business Logic
The Microsoft 365 admin model should avoid using Global Administrator for normal workload administration when specialist roles can handle the task.
