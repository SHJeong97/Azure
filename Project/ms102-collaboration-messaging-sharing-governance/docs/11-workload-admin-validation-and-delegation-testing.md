# Workload Admin Validation and Delegation Testing

## Business Objective
The company needs proof that Microsoft 365 specialist admin roles can manage the correct workloads and that delegated collaboration resources behave as intended for end users.

## What This Batch Validates
- Exchange Administrator workload access
- SharePoint Administrator workload access
- Teams Administrator workload access
- Global Reader oversight access
- Shared mailbox delegation behavior for department users

## Design Logic
Specialist admin roles should perform workload-specific administration without requiring Global Administrator. Business users should be able to use shared mailbox permissions according to the assigned delegation model.

## Business Value
This proves the service-admin design is practical, reduces unnecessary privilege, and supports real collaboration operations.
