# Validation Plan

## What Will Be Validated
- pilot users created successfully
- pilot groups created successfully
- Microsoft 365 licenses assigned correctly
- shared mailboxes created correctly
- department Teams created correctly
- department SharePoint sites created correctly
- external sharing controls captured
- workload admin roles assigned correctly

## Evidence Types
- portal screenshots
- PowerShell / Graph output
- mailbox delegation views
- team membership views
- SharePoint sharing and site views
- written validation notes

## Validation Standard
Every major Microsoft 365 workload control must have at least one screenshot and one written validation note.
