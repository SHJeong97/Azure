# License and Scope Strategy

## Licensing Baseline
This project uses Microsoft 365 E5 for the pilot users who benefit from PIM and Identity Protection.

## Pilot Scope
The project will use a small pilot instead of modeling all 250 users directly.

## Pilot User Types
- emergency access accounts
- privileged admin accounts
- security-response accounts
- risk-test standard users

## Why This Matters
Advanced identity controls should be piloted deliberately so that configuration, validation, and troubleshooting stay controlled and reviewable.
