# Mail Hygiene Baseline

## Goal
Create a practical mail hygiene baseline for DemoCompany using Microsoft Defender policies without overcomplicating the lab.

## Scope
- Custom anti-phishing policy
- Custom inbound anti-spam policy
- Existing default anti-malware policy retained

## Design Decision
Preset security policies were not enabled for this project scope because custom policy behavior needed to remain visible and testable.

## Protected Business Paths
- Executive identities
- HR shared mailbox
- Finance shared mailbox
- Operations shared mailbox
