# Business Scenario

DemoCompany has a working Microsoft 365 tenant with hybrid identity already established, but its Exchange Online administration model is still immature.

Current problems:
- Shared mailbox usage is inconsistent
- Mailbox access is not standardized by role
- Mail flow control is weak
- Anti-phishing and anti-spam posture is not clearly documented
- IT lacks a repeatable validation process for message delivery and mail security controls

Departments in scope:
- Executive
- IT
- HR
- Finance
- Sales
- Operations

Project goal:
Design and implement a realistic Exchange Online administration and mail flow security baseline for a 75-user company using Microsoft 365.

Success criteria:
- Shared mailboxes are designed and assigned correctly
- Recipient administration is documented
- Mail flow rules support business and security needs
- Anti-spam and anti-phishing settings are reviewed and improved
- Message trace and validation prove the configuration works
