# Mail Access Model

## Shared Mailboxes Planned
- hr@democompany1016.onmicrosoft.com
- finance@democompany1016.onmicrosoft.com
- operations@democompany1016.onmicrosoft.com

## Access Logic
- HR shared mailbox: HR team only
- Finance shared mailbox: Finance team only
- Operations shared mailbox: Operations leadership or operations staff only

## Administrative Logic
- Exchange administration should not rely on Global Administrator for routine recipient tasks
- Role-appropriate Exchange administration should be preferred where possible

## Validation Logic
- Shared mailbox access must be tested
- Mail flow must be tested with internal and policy-affected messages
- Message trace should confirm delivery behavior
