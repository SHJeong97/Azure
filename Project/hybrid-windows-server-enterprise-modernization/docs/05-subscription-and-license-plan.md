# Subscription and License Plan

## Current Lab Subscriptions and Trials

The lab uses:

- Azure subscription
- Office 365 E3 trial
- Office 365 E5 trial
- Enterprise Mobility + Security E5 trial

## Azure Subscription Usage

The Azure subscription will be used for:

- Resource groups
- Virtual networks
- Network security groups
- Windows Server virtual machines
- Storage accounts
- Azure Files
- Azure File Sync
- Azure Arc
- Log Analytics
- Recovery Services Vault
- Azure Backup

## Office 365 E3/E5 Usage

Office 365 E3/E5 will be used for:

- Microsoft 365 admin center
- Exchange Online administration
- Mailbox testing
- Shared mailbox testing
- Microsoft 365 custom domain verification
- SPF, DKIM, and DMARC validation
- Outlook/OWA validation
- Teams, SharePoint, and OneDrive testing where needed

## EMS E5 Usage

Enterprise Mobility + Security E5 will be used for:

- Microsoft Entra ID P2
- Conditional Access
- Identity Protection
- Microsoft Entra Application Proxy
- Microsoft Intune
- Device compliance
- Endpoint configuration policies
- Endpoint security policies
- MD-102-related endpoint management scenarios

## Public Domain

Not required in Project 01.

Later, the company will purchase or simulate a public business domain:

summitridgemfg.com

This will be used for:

- Microsoft 365 domain verification
- User sign-in names
- Exchange Online primary email addresses
- SPF, DKIM, DMARC
- Autodiscover
- Application Proxy custom domain planning

## Azure DNS Public Zone

Not required in Project 01.

Later, Azure DNS Public Zone may be used for public DNS hosting if the project chooses Azure-managed public DNS instead of registrar DNS.

## Defender for Cloud

Not required in Project 01.

Paid Defender for Cloud plans should not be enabled until the security-hardening project.
