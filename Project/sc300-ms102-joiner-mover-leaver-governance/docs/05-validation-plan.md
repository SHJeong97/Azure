# Validation Plan

## What Will Be Validated
- User creation and baseline assignment
- Department access group membership
- License inheritance
- Mover old-access removal
- Mover new-access assignment
- Leaver sign-in block
- Leaver license removal
- Leaver access removal from groups and resources

## Evidence Types
- Portal screenshots
- PowerShell / Graph output
- Before-and-after group membership views
- License assignment views
- User account state views
- Validation notes for each lifecycle event

## Validation Standard
Each lifecycle event must have both:
- at least one screenshot
- at least one written validation note
