# Publish Checklist

## Security Review
- No passwords stored in the repository
- No session data or recovery details exposed in screenshots
- No private secrets or tokens included in notes or scripts

## Content Review
- README section order is correct
- Evidence filenames referenced in the README exist
- Validation notes exist for each major build phase
- Usernames, group names, and mailbox names are consistent
- Project title is consistent across files

## Presentation Review
- Architecture diagram exists
- Business value file exists
- Interview talk track exists
- Evidence index exists
- Repo name is professional and consistent
