# Project Scope

## In Scope
- Cloud-only Microsoft Entra ID and Microsoft 365 identity lifecycle model
- Baseline users and department groups
- Joiner workflow
- Mover workflow
- Leaver workflow
- Group-based licensing
- Validation evidence

## Optional Expansion
- Access Reviews
- Lifecycle Workflows
- Additional governance automation

## Out of Scope
- Hybrid identity
- On-prem Active Directory
- Third-party HR integration
