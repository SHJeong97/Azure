# Joiner Workflow

## Business Objective
New hires must receive the right baseline access on day one without requiring one-off manual decisions for each account.

## Joiner Design
Joiners are onboarded through:
- account creation
- baseline Microsoft 365 E3 licensing
- department group assignment
- shared mailbox access assignment where required
- validation evidence collection

## Joiner Scenarios in This Project
- Chloe Adams joins the Sales department
- Jackson Reed joins the Operations department

## Control Logic
Each joiner must receive:
- the baseline employee license
- the correct department access group
- the correct department shared mailbox access
- inclusion in the joiner validation group for evidence tracking

## Business Value
A standardized joiner process reduces onboarding delays, access inconsistency, and support overhead while improving auditability.
